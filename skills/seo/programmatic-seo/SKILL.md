# Programmatic SEO

## Core Principles

1. **Unique Value Per Page** - Every page must provide value specific to that page
2. **Proprietary Data Wins** - Your own data > public data
3. **Clean URL Structure** - Subfolders, not subdomains
4. **Quality Over Quantity** - Better 100 great pages than 10,000 thin ones

## The 12 Playbooks

| Playbook | Pattern | Example |
|----------|---------|---------|
| Templates | "[Type] template" | "resume template" |

## Implementation Framework

1. Keyword pattern research
2. Data requirements and sources
3. Template design with unique content per page
4. Internal linking architecture (hub and spoke)
5. Indexation strategy and sitemaps

## Quality Checks

- [ ] Each page provides unique value
- [ ] Answers search intent
- [ ] Unique titles and meta descriptions
- [ ] Proper heading structure
- [ ] Schema markup implemented
- [ ] Connected to site architecture
- [ ] In XML sitemap
