# AI SEO (AI Search Optimization)

## How AI Search Works

| Platform | Source Selection |
|----------|----------------|
| Google AI Overviews | Top-ranking pages |
| ChatGPT (with search) | Wide range, high authority |
| Perplexity | Authoritative, recent, well-structured |
| Gemini | Google index + Knowledge Graph |
| Claude | Brave Search |

## The Three Pillars

### 1. Structure - Make Content Extractable
- Definition blocks for "What is X?"
- Step-by-step for "How to X"
- Comparison tables for "X vs Y"
- FAQ blocks for common questions
- Keep answer passages 40-60 words

### 2. Authority - Make Content Citable
- Cite sources with links (+40% visibility)
- Include statistics with sources (+37%)
- Add expert quotes with credentials (+30%)
- Show "Last updated" dates
- Use authoritative tone (+25%)

### 3. Presence - Be Where AI Looks
- Wikipedia mentions (7.8% of ChatGPT citations)
- Reddit discussions
- Industry publications
- Review sites (G2, Capterra)
- YouTube content

## Content Types That Get Cited Most

| Type | Citation Share |
|------|:------------:|
| Comparison articles | ~33% |
| Definitive guides | ~15% |
| Original research/data | ~12% |
| Best-of/listicles | ~10% |
| How-to guides | ~8% |

## Common Mistakes

- Keyword stuffing (reduces visibility by 10%)
- Blocking AI bots (GPTBot, PerplexityBot, ClaudeBot)
- No structured data/schema markup
- Gating all content behind login
- Undated content
