# SEO Audit

## Audit Framework

Priority order:
1. Crawlability & Indexation
2. Technical Foundations
3. On-Page Optimization
4. Content Quality
5. Authority & Links

## Technical SEO Audit

### Crawlability
- robots.txt: Check for unintentional blocks
- XML Sitemap: Exists, accessible, submitted to Search Console
- Site Architecture: Important pages within 3 clicks of homepage
- Crawl Budget: Parameterized URLs, faceted navigation, infinite scroll

### Indexation
- Index status via site:domain.com
- Noindex tags on important pages
- Canonical tags pointing correctly
- Redirect chains/loops
- Soft 404s and duplicate content

### Core Web Vitals
- LCP < 2.5s, INP < 200ms, CLS < 0.1
- Server response time (TTFB)
- Image optimization
- JavaScript execution
- CDN and caching

### Mobile & Security
- Responsive design
- HTTPS across entire site
- Valid SSL certificate
- No mixed content

## On-Page SEO Audit

### Title Tags
- Unique per page, 50-60 chars
- Primary keyword near beginning

### Meta Descriptions
- Unique per page, 150-160 chars
- Clear value proposition and CTA

### Heading Structure
- One H1 per page
- Logical hierarchy (H1 > H2 > H3)

### Content Optimization
- Keyword in first 100 words
- Related keywords naturally used
- Sufficient depth for topic
- Answers search intent

### Image Optimization
- Descriptive file names
- Alt text on all images
- Modern formats (WebP)
- Lazy loading

### Internal Linking
- Important pages well-linked
- Descriptive anchor text
- No broken internal links

## Output Format

### Audit Report Structure
1. Executive Summary (overall health, top 3-5 priorities, quick wins)
2. Technical SEO Findings (issue, impact, evidence, fix, priority)
3. On-Page Findings (same format)
4. Content Findings (same format)
5. Prioritized Action Plan
