## Agent 更新义务

**当 agent 遇到以下情况时，必须更新本文档：**
- 遇到错误并解决时 → 记录错误、原因和解决方案
- 发现项目中的潜在风险时 → 记录风险和预防建议
- 踩坑后总结出新经验时 → 添加到"常见陷阱"表格
- 任务完成后回顾发现可改进点时 → 记录改进建议
